# [Cross2D](Cross2D)
##### >[Psychtoolbox](Psychtoolbox)>[PsychCal](PsychCal)

output = [Cross2D](Cross2D)(v1,v2)  
  
Produce a matrix of 2D column vectors that cross all  
combinations of the passed input vectors.  
  
See also meshgrid, ndgrid.
